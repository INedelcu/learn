Variance of the estimator, the CLT & error bars

04_variance_and_clt.cpp — why the Monte Carlo error falls as 1/√N (now proved, not just observed), and how a renderer attaches an honest “±” to every result and decides a pixel has converged.

Example 03 produced one number per run. But that number is itself a random variable: run the estimator again with fresh samples and you get a slightly different answer. This file studies that randomness head-on — its size (variance), its shape (the Central Limit Theorem), and the error bar and confidence interval you can build from a single run. It then tests the theory by launching 100,000 independent estimators and checking they behave exactly as predicted.

One-sentence summary. The estimate $F_N$ is unbiased with variance $\operatorname{Var}[Y]/N$, so its standard deviation — the standard error — is $\sigma_Y/\sqrt N$ (this is the $1/\sqrt N$ law, proved); the Central Limit Theorem makes $F_N$ approximately $\mathrm{Normal}(I,\sigma_Y^2/N)$, so $F_N\pm1.96\,\mathrm{SE}$ is a 95% confidence interval — and you can estimate $\mathrm{SE}$ from the very same samples, getting an error bar for free.

1. The estimate is a random variable

We reuse the integral from example 03: $f(x)=e^x$ on $[0,1]$, whose exact value is $I=e-1\approx1.7182818$. Writing $Y_i=f(X_i)/p(X_i)$ for each sampled term, the estimator is their average

$$F_N=\frac1N\sum_{i=1}^N Y_i .$$

Each run uses different random $X_i$, so each run lands on a slightly different $F_N$. Collect the estimates from many runs and they form a cloud around the true value — the sampling distribution of the estimator. Everything below is about the size and shape of that cloud.

2. The size of the cloud: variance falls as 1/N

Three facts pin down the cloud. The estimator is an average of $N$ independent copies of $Y$, so:

It is unbiased — centered on the right answer:

$$\mathbb E[F_N]=\frac1N\sum_{i=1}^N\mathbb E[Y_i]=\mathbb E[Y]=I.$$

Its variance falls as $1/N$. Variance of a sum of independent things is the sum of their variances, and pulling the $1/N$ out of the average squares it:

$$\operatorname{Var}[F_N]=\frac1{N^2}\sum_{i=1}^N\operatorname{Var}[Y_i]=\frac{N\,\operatorname{Var}[Y]}{N^2}=\frac{\operatorname{Var}[Y]}{N}.$$

So its standard deviation falls as $1/\sqrt N$:

$$\operatorname{SD}[F_N]=\frac{\sigma_Y}{\sqrt N}\equiv\mathrm{SE},\qquad \sigma_Y=\sqrt{\operatorname{Var}[Y]} .$$
This is the law from example 02 — now derived. Earlier we just watched the error fall as $1/\sqrt N$ and saw error$\times\sqrt N$ flatten to a constant. That constant has a name now: it is $\sigma_Y$, the standard deviation of a single sample. The error of an $N$-sample estimate is $\sigma_Y/\sqrt N$. Variance reduction (examples 06–09) is the project of shrinking $\sigma_Y$.

3. The standard error, and a free error bar

The catch: $\sigma_Y$ depends on $\operatorname{Var}[Y]$, which itself depends on the integral we don’t know. The fix is to estimate $\sigma_Y$ from the very same samples, using the Bessel-corrected sample variance from example 01. With sample variance $s^2=\tfrac1{N-1}\sum(Y_i-F_N)^2$, the estimated standard error is

$$\widehat{\mathrm{SE}}=\frac{s}{\sqrt N}=\sqrt{\frac{s^2}{N}} .$$

4. The shape of the cloud: the Central Limit Theorem

Knowing the cloud’s center and width is not enough to make a 95% statement — for that we need its shape. The Central Limit Theorem supplies it: as $N$ grows, the distribution of an average approaches a Normal (Gaussian), no matter what shape the individual $Y$ has. Here $Y=e^X$ is quite skewed, yet $F_N$ becomes a clean bell:

$$F_N\;\xrightarrow{\;\text{large }N\;}\;\mathrm{Normal}\!\left(I,\ \frac{\sigma_Y^2}{N}\right).$$

Build the sampling distribution yourself. Each click runs more estimators (each an average of $K$ samples) and drops them into the histogram. Toggle the Normal curve to see how well $\mathrm{Normal}(I,\sigma_Y^2/K)$ matches, and drag $K$ to watch the bell narrow as $1/\sqrt K$ while keeping its shape.

Why 1.96?

A Normal places a fixed fraction of its mass within a given number of standard deviations of its center — the “68–95–99.7” rule. Exactly 95% lies within $\pm1.96\sigma$, which is where the magic number comes from.

5. Confidence intervals — and what “95%” really means

Put the width (§3) and the shape (§4) together. Since $F_N$ is approximately $\mathrm{Normal}(I,\mathrm{SE}^2)$, the interval

$$\big[\,F_N-1.96\,\widehat{\mathrm{SE}}\;,\;\; F_N+1.96\,\widehat{\mathrm{SE}}\,\big]$$

is an approximate 95% confidence interval for $I$. The subtle part is the interpretation: $I$ is a fixed number, not random. What is random is the interval — it jumps around with each run. The claim is frequentist:

What “95% confidence” means. It is not “there’s a 95% chance $I$ is in this particular interval.” It is: if you repeated the whole experiment many times, about 95% of the intervals you build this way would contain the true $I$. The interval is the random thing; the coverage is a long-run frequency.

6. Testing the theory: 100,000 estimators

Part 2 of the file is an empirical audit. It runs many independent estimators (each $K=1000$ samples), and checks two predictions at once: that the spread of all the estimates equals $\sigma_Y/\sqrt K$, and that each estimator’s self-built 95% interval covers the truth about 95% of the time. Run it below — each row is one estimator’s interval, green if it caught $I$, red if it missed; the running coverage settles on 95%.

Why this matters for rendering. This is what lets a path tracer be honest about noise. The per-pixel variance, accumulated online while sampling, yields a standard error; the CLT turns it into a confidence band. That band is exactly what adaptive samplers watch to decide “this pixel is converged, stop spending samples here” and what denoisers and noise-threshold stopping criteria are built on.

7. The code

The complete sample on GitHub: 04_variance_and_clt.cpp.

Part 1 is one Welford pass producing the estimate, its standard error, and the 95% interval together:

double mean = 0.0, m2 = 0.0; long long n = 0;
for (long long i = 0; i < 10000000; ++i) {
    double y = f(rng.NextFloat());     // Y = f(X)/p(X), p = 1 on [0,1]
    ++n;
    double delta = y - mean; mean += delta / n; m2 += delta * (y - mean);  // Welford
    if (n == nextReport) {
        double var = m2 / (n - 1);     // sample variance of Y   (Bessel)
        double se  = sqrt(var / n);    // standard error of the mean
        printf("%lld  %.7f  SE=%.7f  CI=[%.6f, %.6f]\n",
               n, mean, se, mean - 1.96*se, mean + 1.96*se);
        nextReport *= 10;
    }
}

Part 2 runs kNumEstimates = 100000 estimators of kPerEstimate = 1000 samples each, measures the spread of the estimates, and counts coverage — then compares against the analytic prediction $\sigma_Y/\sqrt K$ using the exact $\operatorname{Var}[Y]=\tfrac{e^2-1}{2}-(e-1)^2$:

int covered = 0; double gMean = 0, gM2 = 0;           // stats OF the estimates
for (int e = 0; e < kNumEstimates; ++e) {
    double m = 0, s2 = 0;                              // one estimator (Welford)
    for (int i = 0; i < kPerEstimate; ++i) {
        double y = f(rng.NextFloat());
        double d = y - m; m += d/(i+1); s2 += d*(y-m);
    }
    double se = sqrt((s2/(kPerEstimate-1)) / kPerEstimate);
    if (fabs(m - I) <= 1.96*se) ++covered;             // did the CI cover I?
    double d = m - gMean; gMean += d/(e+1); gM2 += d*(m-gMean);
}
double spread = sqrt(gM2/(kNumEstimates-1));           // ~ sigma_Y/sqrt(K)
double predSE = sqrt(((kE*kE-1)/2 - I*I) / kPerEstimate);

Typical output: mean of estimates $\approx I$, spread $\approx\,$predicted SE $\approx0.0156$, and coverage $\approx95\%$ — the estimator behaves exactly like the Normal the CLT promises.

8. Takeaways

  • The estimate is random. Its sampling distribution is centered on $I$ (unbiased) with variance $\operatorname{Var}[Y]/N$.
  • $1/\sqrt N$, proved. $\operatorname{SD}[F_N]=\sigma_Y/\sqrt N$ — the standard error. This is the law example 02 only observed; variance reduction shrinks $\sigma_Y$.
  • Free error bar. Estimate $\sigma_Y$ from the same samples (Bessel sample variance) to get $\widehat{\mathrm{SE}}=s/\sqrt N$ in a single Welford pass.
  • CLT gives the shape. For large $N$, $F_N\approx\mathrm{Normal}(I,\sigma_Y^2/N)$ regardless of the integrand, so $\pm1.96\,\widehat{\mathrm{SE}}$ is a 95% interval.
  • Coverage is a long-run frequency. Over many runs ~95% of the intervals contain $I$; verified empirically at ~95%. This is the machinery behind honest noise estimates and adaptive sampling.
