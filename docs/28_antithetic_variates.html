Antithetic variates: pair each sample with its mirror

28_antithetic_variates.cpp — for every uniform sample u, also evaluate at 1−u and average the pair. When the integrand trends one way, the mirror cancels its swing — variance reduction for one extra subtract.

Here is the cheapest variance-reduction trick in the book. For every uniform draw $u$, also use its mirror $1-u$, and average the two evaluations. If the integrand is monotonic, the mirror is its mood-opposite — when $f(u)$ is high, $f(1-u)$ is low — so the pair-average barely moves while the plain estimate swings. One negate per sample, and the variance can drop by tens of times. The one caveat: for a function symmetric about $\tfrac12$ the mirror is a clone, not an opposite, and the trick does nothing (or worse).

One-sentence summary. The pair-average $\tfrac12(f(u)+f(1-u))$ has variance $\tfrac12\operatorname{Var}(f)\,(1+\rho)$ with $\rho=\operatorname{corr}(f(u),f(1-u))$, so at equal evaluation budget antithetic sampling beats plain Monte Carlo by $1/(1+\rho)$ — a big win when the integrand is monotonic (mirror anti-correlated, $\rho<0$; for a linear $f$, $\rho=-1$ and the variance vanishes), and no help when it is symmetric ($\rho=+1$).

1. The mirror trick

Plain Monte Carlo draws independent uniforms and averages $f$. Antithetic sampling pairs each $u$ with $1-u$ — also uniform, so still unbiased — and averages the pair:

$$\hat I_{\text{anti}}=\frac{1}{M}\sum_{k=1}^{M}\frac{f(u_k)+f(1-u_k)}{2}.$$

Its leverage is the variance of one pair-average. With $A=f(u)$ and $B=f(1-u)$, both with the same variance $\operatorname{Var}(f)$ (since $1-u$ is uniform too),

$$\operatorname{Var}\!\Big[\tfrac{A+B}{2}\Big]=\tfrac14\big(\operatorname{Var}A+\operatorname{Var}B+2\operatorname{Cov}(A,B)\big) =\tfrac12\operatorname{Var}(f)\,(1+\rho),\qquad \rho=\operatorname{corr}\big(f(u),f(1-u)\big).$$

A pair costs two evaluations, so to compare fairly against plain MC at the same budget, the variance per evaluation is twice that, $\operatorname{Var}(f)(1+\rho)$, versus $\operatorname{Var}(f)$ for plain. The ratio is the whole story:

$$\boxed{\ \text{variance reduction}=\frac{1}{1+\rho}\ }$$

If the mirror is anti-correlated ($\rho<0$) you win; the more negative $\rho$, the bigger the win. Below, $f(u)$ and its mirror $f(1-u)$, and their average — flat for a monotonic integrand, unchanged for a symmetric one:

2. When it helps — and when it doesn't

The sign of $\rho$ is set by the integrand's shape. A monotonic $f$ makes $f(u)$ and $f(1-u)$ move in opposite directions — large $u$ gives large $f(u)$ but small $f(1-u)$ — so $\rho<0$. In the limit of a linear $f(x)=a+bx$, the pair-average $\tfrac12\big[(a+bu)+(a+b(1-u))\big]=a+\tfrac b2$ is a constant: $\rho=-1$ and the variance is exactly zero. A symmetric $f$ with $f(u)=f(1-u)$ gives $\rho=+1$, so the reduction $1/(1+\rho)=\tfrac12$ — antithetic doubles the variance per evaluation, strictly worse. The C++ run at equal budgets:

3. In the renderer

Antithetic pairing is most useful on the monotonic 1-D maps inside a sampler: the radial inverse-CDF of a light falloff, a time sample for motion blur, a 1-D scattering-distance draw. Mirror the uniform that feeds such a map and the paired estimate cancels the linear trend almost for free. It is not a blanket win — a renderer's full path integrand is high-dimensional and rarely monotonic in the raw uniforms — which is why it is a targeted tool, not a default like stratification. The cost is one extra evaluation and a subtract; the discipline is to apply it only where the integrand actually trends, and never where it is symmetric.

4. The code

The complete sample on GitHub: 28_antithetic_variates.cpp.

One uniform feeds two evaluations; the pair-average is the per-pair estimate.

double u = rng.NextFloat();
double a = f(u), b = f(1.0 - u);                 // the sample and its mirror
double pairAvg = 0.5 * (a + b);                  // one estimate, two evaluations
// Var[pairAvg] = 0.5 * Var(f) * (1 + corr(f(u), f(1-u)))
// vs plain MC at equal budget: reduction = 1 / (1 + rho).
//   monotonic f -> rho < 0 (win);  linear f -> rho = -1 (zero variance);
//   symmetric f -> rho = +1 (0.5x, worse).
Takeaway. Pair every uniform $u$ with its mirror $1-u$ and average: unbiased, and the pair-average has variance $\tfrac12\operatorname{Var}(f)(1+\rho)$, so at equal cost antithetic sampling beats plain MC by $1/(1+\rho)$. It is a large, near-free win for monotonic 1-D maps (linear integrands hit zero variance), and a $0.5\times$ penalty for symmetric ones — a sharp, targeted tool, not a default.
