Delta tracking in heterogeneous media

33_delta_tracking.cpp — fog and clouds have an extinction that varies along the ray, so the transmittance has no invertible form. Delta tracking pads the medium with fictitious "null" collisions to a constant rate you can sample, then thins them away — unbiased, no integral to invert.

Sampling the distance to a collision in a homogeneous medium was easy (example 05): transmittance $e^{-\sigma t}$ inverts to $t=-\ln(1-u)/\sigma$. But fog, smoke and clouds have a heterogeneous extinction $\sigma(x)$ that varies along the ray, so the transmittance $T(t)=\exp(-\!\int_0^t\sigma)$ has no closed-form inverse to sample. Delta tracking (Woodcock 1965) sidesteps the integral entirely with a fictitious-collision trick that turns the varying medium back into simple exponential free flights.

One-sentence summary. Pad the medium with null collisions so the total event rate is a constant majorant $\sigma_{\max}\ge\sigma(x)$ everywhere; then free-flight distances are exponential at rate $\sigma_{\max}$ (easy), and at each tentative collision you accept it as real with probability $\sigma(x)/\sigma_{\max}$ — by Poisson thinning the accepted events have rate exactly $\sigma(x)$, so collision distances follow the true pdf $\sigma(t)\,T(t)$ and the survival probability is the transmittance, all without inverting the optical depth.

extinction $\sigma(x)$ majorant $\sigma_{\max}$ real collision (accept) null collision (reject) pdf $\sigma(t)T(t)$

1. A medium you can't invert

A ray through a medium collides with probability set by the extinction $\sigma$: the chance of surviving to distance $t$ without a collision is the transmittance

$$T(t)=\exp\!\Big(-\!\int_0^t\sigma(s)\,ds\Big),$$

and the collision distance has pdf $\sigma(t)T(t)$. When $\sigma$ is constant this is the exponential of example 05, trivially invertible. When $\sigma(x)$ varies — as it must for any interesting cloud — the optical depth $\int_0^t\sigma$ is some non-trivial function and its exponential has no inverse you can evaluate per-sample. Rejection-style marching with tiny steps is biased and slow. We need a way to sample the exact distribution without ever inverting that integral.

2. Pad to a constant rate, then thin

Choose a majorant $\sigma_{\max}\ge\sigma(x)$ for the whole ray, and imagine filling the "missing" extinction $\sigma_{\max}-\sigma(x)$ with fictitious particles that scatter into nothing — null collisions. The padded medium has constant total rate $\sigma_{\max}$, so free flights are plain exponentials. March:

  1. step $t\mathrel{+}=-\ln(1-u)/\sigma_{\max}$ — an exponential free flight at the majorant rate;
  2. at the tentative collision, accept it as real with probability $\sigma(t)/\sigma_{\max}$, otherwise it was a null collision — ignore it and keep marching.

Why the accepted distances are exact. The tentative collisions are a Poisson process of constant rate $\sigma_{\max}$. Marking each independently "real" with probability $\sigma(t)/\sigma_{\max}$ is exactly thinning a Poisson process, and the thinned process has rate $\sigma_{\max}\cdot\dfrac{\sigma(t)}{\sigma_{\max}}=\sigma(t)$ — the medium's true extinction. So the real collisions occur at rate $\sigma(t)$, their first-arrival distance has pdf $\sigma(t)\,T(t)$, and the chance of reaching $L$ with no real collision is $T(L)$ — the transmittance. The null collisions are precisely the padding that makes the rate constant, and thinning removes them in expectation. No optical-depth inverse anywhere.

3. It hits the true distribution

The same majorant march also gives transmittance cheaply by ratio tracking: instead of accepting or rejecting, carry a weight and multiply it by $\big(1-\sigma(t)/\sigma_{\max}\big)$ at every tentative collision. Its expectation over the Poisson($\sigma_{\max}$) process is $\exp\!\big(-\!\int_0^L\sigma_{\max}\cdot\tfrac{\sigma}{\sigma_{\max}}\big)=\exp(-\!\int\sigma)=T(L)$ — the transmittance, with much lower variance than the accept/reject fraction. The C++ run: delta tracking $0.018299$, ratio tracking $0.018284$, both on the analytic $0.018316$.

4. In the renderer

Delta tracking is how production volume renderers march rays through clouds, smoke, and any voxelized or procedural density — the density grid supplies $\sigma(x)$, a precomputed max supplies $\sigma_{\max}$, and the accept/reject loop above samples scatter events without ever integrating the field. Ratio tracking gives the shadow-ray transmittance for next-event estimation (example 24) through the volume. The PathTracingDemo glass shader only needs the homogeneous case — Beer–Lambert $e^{-\sigma t}$ along the transmitted segment (example 22) — which is exactly delta tracking's $\sigma=\text{const}$ limit, where every tentative collision is real and the free flight is the closed-form exponential. The tighter $\sigma_{\max}$ hugs $\sigma$, the fewer null collisions are wasted — the one knob that governs delta tracking's cost.

5. The code

The complete sample on GitHub: 33_delta_tracking.cpp.

Exponential free flights at the majorant rate; accept as real with probability $\sigma/\sigma_{\max}$.

// delta tracking: sample the collision distance in a heterogeneous medium
double t = 0.0; bool collided = false;
for (;;) {
    t += -std::log(1.0 - rng.NextFloat()) / sMax;       // free flight at rate sigma_max
    if (t >= L) break;                                  // survived the slab (transmitted)
    if (rng.NextFloat() < Sigma(t) / sMax) { collided = true; break; }   // real collision
    // else: null collision -- ignore and keep marching
}

// ratio tracking: transmittance estimate, no accept/reject -- carry a weight
double w = 1.0;
for (t = 0;;) {
    t += -std::log(1.0 - rng.NextFloat()) / sMax;
    if (t >= L) break;
    w *= (1.0 - Sigma(t) / sMax);                        // E[w] = exp(-integral sigma) = T(L)
}
Takeaway. A heterogeneous medium's transmittance $\exp(-\!\int\sigma)$ can't be inverted to sample collisions directly. Delta tracking pads it with null collisions to a constant majorant $\sigma_{\max}$, sampling exponential free flights and accepting each tentative collision as real with probability $\sigma/\sigma_{\max}$. By Poisson thinning the accepted events have the true rate $\sigma(t)$, so distances follow $\sigma(t)T(t)$ and survival gives the transmittance — verified to match the analytic values. It is how renderers march clouds; Beer–Lambert glass is the $\sigma=\text{const}$ limit.
